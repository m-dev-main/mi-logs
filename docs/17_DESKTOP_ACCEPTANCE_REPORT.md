## Security Notes

- The desktop renderer uses `#/admin` hash routing for packaged `file://` loading.
- The desktop bridge is compiled only into `build:desktop`; public release builds compile it out.
- Desktop admin API access requires the Electron IPC bridge, the per-launch desktop-control secret, and the native biometric lock.
- Public release artifacts are generated by `pnpm release`, so desktop-only admin paths are not exported to `releases/latest/public`.
